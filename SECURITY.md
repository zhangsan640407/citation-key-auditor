# Security Policy

Citation Key Auditor is a local command-line tool. It does not send manuscript
content, citation keys, or BibTeX files to external services.

## Supported versions

Security fixes are accepted for the latest released version.

## Reporting

Please open a private security advisory on GitHub if the repository has
advisories enabled. If not, open an issue with a minimal description that does
not expose sensitive files.

Useful reports include:

- command invocation
- operating system
- Python version
- minimal input that reproduces the issue
