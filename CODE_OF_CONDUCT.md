# Code of Conduct

This project welcomes practical, respectful collaboration.

Participants are expected to:

- discuss technical disagreements with evidence and reproducible examples
- avoid harassment, personal attacks, and exclusionary behavior
- respect maintainers' time by keeping reports specific and actionable
- assume public discussion may be read by new contributors

Maintainers may remove abusive comments, close bad-faith issues, or block
participants who repeatedly disrupt the project.
